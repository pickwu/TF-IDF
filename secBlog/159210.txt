在Linux平台安装
安装命令如下：
git clone https://github.com/Manisso/fsociety.git cd fsociety && python fsociety.py
工具的安装视频如下：
视频地址：https://www.youtube.com/watch?v=t3uYpMrK2EU
Android平台安装视频：https://www.youtube.com/watch?v=JwK5oOBjpgQ&t=160s
在Windows平台安装
1. 下载Linux Bash，例如Cygwin
2. 下载Python2.7
3. 或者你也可以使用Google云控制台（CloudShell）安装。
